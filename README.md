# Snake Game

## How to install game?

### You can follow this [link](https://drive.google.com/open?id=1bl9MCDobRdGB9YFKxSmBrf0U4dL1cUkj) and download the game.

#### 1. Go to the [link](https://drive.google.com/open?id=1bl9MCDobRdGB9YFKxSmBrf0U4dL1cUkj) and download exe file
#### 2. Run the game
![screenshot](https://i.ibb.co/1szYzZB/1Step1.png)

#### 3. Press 'w', 'a', 's', 'd' or arrows to start the game.
![screenshot](https://i.ibb.co/JkKH9GB/1Step2.png)

#### 4. Use'w', 'a', 's', 'd' or arrows to move the snake.
![screenshot](https://i.ibb.co/BK0qF8M/1Step3.png)


### You can also download the source code and compile the program into a Visual Studio.

#### 1. Click on this [link](https://github.com/KoTuK777/SnakeGame/archive/master.zip) to download zip file with code.
![screenshot](https://i.ibb.co/1szYzZB/1Step1.png)

### 2. Unzip the archive
![screenshot](http://b1.org/images/help/open_zip_archive_context/open_zip_archive_2_extrhere.png)

### 3. Open sln file.
![screenshot](https://i.ibb.co/3mT21Cq/Step2.png)

### 4. Then click on "SnakeGame".
![screenshot](https://i.ibb.co/tcs4J5t/Step3.png)

### 5. Open "Headers Files" to see the class code and "Source Files" to see main function.
